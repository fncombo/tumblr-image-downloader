Locations

Define a list of custom locations inside your downloads folder that you will be able to quickly choose from when saving an image.

Separate folders inside folders with a / forward slash. If the folder doesn't exist, it will automatically be created.

Downloaded Images

Tumblr Image Downloader automatically remembers all the images you have downloaded and puts a ✓ tick next to the download button.

This will permanently forget all the images you have downloaded so far.

Confirmation

Toggle whether to bring up a confirmation dialog when downloading an already downloaded image.

Changing settings will affect any currently open pages.

Please remember to respect the images' owners and follow their rules and usage guidelines :) This extension is only supposed to enable you to collect those images for yourself, not redistribute them without the original author's permission.
